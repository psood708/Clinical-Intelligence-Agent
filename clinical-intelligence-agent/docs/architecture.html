Agent Layer (LangGraph State Machine)

01 · Extract
Extractor
⚡ qwen3.5:9b · Ollama
Structured clinical entity extraction. Outputs validated Pydantic schema: meds, conditions, labs, risk flags.
local · $0
02 · Retrieve
Retriever
🔍 HNSW + BioBERT
Searches clinical corpus. BioBERT embeddings → HNSW ANN (your project) → Cohere Rerank 3.5 post-pass.
local + cohere free
03 · Verify
Verifier
🧠 gemma4:e2b · Ollama
Challenges extraction vs retrieved evidence. Flags hallucinations. Triggers loop-back on critical issues.
local · $0
04 · Synthesize
Synthesizer
🚀 llama3.3-70b · Groq
Final clinical summary with source citations. Routes to Groq (315 tok/s) or falls back to local Ollama.
groq free tier
↻
Self-correction loop — if Verifier finds critical flags, pipeline loops back to Extractor with correction hints. Max 2 iterations. This drops hallucination rate from ~12% → ~4%.
